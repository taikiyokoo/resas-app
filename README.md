# 都道府県人口遷移グラフSPA

## 仕様

*都道府県を選択すると選択された都道府県の人口数遷移をRESAS APIから取得してグラフに表示する。*

*「総人口」、「老年人口」、「年少人口」、「生産年齢人口」で切り替えられる。*
